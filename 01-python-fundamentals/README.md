# Student Performance Analyzer

A simple Python project that analyzes student performance based on marks.

## Features
- Add multiple students with marks
- Calculate grades
- Find average marks
- Identify top performer

## Concepts Used
- Python dictionaries
- Functions
- Loops and conditionals
- Built-in functions

## How to Run
1. Open `main.py`
2. Run the program
3. Enter student details when prompted

## Sample Output
Displays student grades, average marks, and top performer.

## Why This Project
This project demonstrates my understanding of Python fundamentals such as
data structures, functions, and control flow through a real-world example.
