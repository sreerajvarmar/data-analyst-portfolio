# Superstore Sales Analysis

This project performs exploratory data analysis on a real-world Superstore sales dataset to extract meaningful business insights.

## Objectives
- Analyze overall sales and profit performance
- Identify top-performing product categories
- Compare sales performance across regions
- Understand monthly sales trends

## Dataset
- Source: Kaggle (Global Superstore Dataset)
- File used: superstore.csv

## Tools & Libraries
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data cleaning and preprocessing
- Date format handling
- Category-wise sales analysis
- Region-wise sales analysis
- Monthly sales trend analysis
- Data visualization for insights

## Key Insights
- Sales performance varies significantly across categories
- Certain regions contribute more to total revenue
- Monthly trends indicate seasonal variations in sales

## How to Run
1. Open `analysis.ipynb`
2. Run cells sequentially
3. Review outputs and visualizations

## Conclusion
This project demonstrates practical data analysis skills including data cleaning, aggregation, and visualization using Python.
